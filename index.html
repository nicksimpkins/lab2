Course Catalog Description:

It kind of cut off at the end and I'm not sure what's supposed to be there but basically the course is about learning HTML, CSS and JavaScript each with their own specialty in web design

Students Completing this course will be able to:

    Recognize HTML, CSS and JS markup code in a web page
    Select proper mark-up tags or code to acheive a certain result
    Identify improper code
    Produce modern standards compliant web pages
    Deploy web pages to a public server

Course Objectives

At the conclusion of this course, each successful student will be able to:

    Explain the client and server archetecture of the internet and related web technologies
    Use a basic text editor and other software tools to create web pages using HTML, CSS, JS, jQuery and images
    Deploy web pages to a linux based web server using sftp
